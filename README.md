# JogoCyberSecurity

Este repositório contém a aplicação do Quiz dentro da pasta `Quiz`.

Como rodar localmente:

1. Entre na pasta do projeto web:

```bash
cd Quiz
```

2. Instale dependências e rode em modo de desenvolvimento:

```bash
npm install
npm run dev
```

Observações:
- A pasta `Quiz` já contém o `package.json` e a app React/Vite.
- Adicionei um `.gitignore` na raiz para evitar subir dependências e arquivos gerados.
