# CYBER QUIZ — Front-End (MVP)

Projeto front-end baseado no PRP fornecido — jogo de dedução social com tema cyberpunk.

Principais pontos:
- React + Vite
- Context API para estado global (simulado)
- Host panel, Player panel, Chat, Votação básica

Como rodar:

```bash
npm install
npm run dev
```

Arquivos principais:
- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/GameContext.jsx`
- `src/components/*`

Observações:
- MVP front-end apenas; multiplayer simulado em estado local.
- Para expandir: adicionar identificação de usuário, roteamento por sala e persistência.
